Приложение скачивает HTML-страницу посредством HTTP-запроса, сохраняет загруженный файл на жесткий диск компьютера, подсчитывает количество уникальных слов на странице и выводит статистику по словам в консоль;
Работа приложения основана на следующем алгоритме;
Имеется входная строка с адресом web-страницы. Приложение скачивает web-страницу, сохраняет ее на жесткий диск;
Затем текстовый контент, который есть на ней, разбивается на слова и подсчитывается количество нахождений на странице каждого уникального слова;
Разделителями слов считаются следующие символы: {' ', ',', '.', '!', '?', '-', '-','"', ';', ':', '[', ']', '(', ')', '\n', '\r', '\t'};
Программа обеспечивает корректную обработку исключительных ситуаций (exceptions);
Все исключительные ситуации в программе реализованы в виде unchecked exception (RunTimeException). В случае появления любого RunTimeException в любом месте программы
оно перехватывается и выводит в консоль причину возникновения ошибки. В консоли всегда можно увидеть текущее состояние ошибки;
Используются стандартные классы;
5.  Приложение должно быть написано на языке Java 8;

Для получения неоходимого jar-файла в системе необходимо установить сборщик maven.
В операционной системе Debian (Ubunty) введите команду:
# apt-get install maven;
запуситите сборку неоходимого jar файла в директории,к которой расположен файл pom.xml, описывающий процесс сборки приложения.
$ mvn clean install;

Необходимый файл расположен в корневой директории проекта /htmlParser/pom.xml
В результате сборки в папке /htmlParser/target/ образуется файл приложения HTML-parser-1.0-SNAPSHOT-jar-with-dependencies.jar, который является файлом приложения,
содержащего в себе все необходимые библиотеки.
